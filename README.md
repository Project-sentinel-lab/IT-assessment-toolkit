IT Assessment Toolkit

Una raccolta di modelli professionali per attività di analisi dei sistemi informativi e cybersecurity. Il repository include template in Markdown per Asset Discovery, Network Assessment, Vulnerability Assessment, Risk Assessment, Hardening Checklist, Incident Response e documentazione tecnica, pensati per supportare assessment aziendali, audit e progetti di miglioramento della sicurezza IT.

Obiettivi

- Standardizzare la raccolta delle informazioni durante gli assessment.
- Documentare infrastrutture, asset e configurazioni di rete.
- Supportare analisi dei rischi e attività di remediation.
- Fornire una base riutilizzabile per audit, consulenze e progetti IT.
- Creare documentazione tecnica chiara, coerente e versionabile tramite Git.

Destinatari

- Analisti di Sistemi Informatici
- Sistemisti
- Consulenti IT
- Cybersecurity Analyst
- IT Manager
- Studenti e professionisti del settore
